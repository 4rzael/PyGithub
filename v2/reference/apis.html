From Github API v3 to PyGithub¶

Here are the 286 end points I’m aware of, and if/where they are implemented in PyGithub. 55 end points are not yet implemented, and I don’t plan to implement 13 end points for reasons described below. If something is not listed here, please open an issue with a link to the corresponding documentation of Github API v3.
