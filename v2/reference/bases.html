Base and utility classes¶

PyGithub.Version = '2.0.0-alpha.3'¶

A string representation of the version of PyGithub as described by semantic versionning.

PyGithub.BlockingBuilder¶

An alias of Builder to make it easy to instanciate.

Session builder¶

class PyGithub.Blocking._builder.Builder¶

Class used to configure the GitHub session. See the User Guide for an introduction.

Login(login, password)¶

Use basic authentication via username and password.

OAuth(token)¶

Use OAuth authentication.

Enterprise(netloc)¶

Use the GitHub Enterprise server with domain name or ip address netloc.

PerPage(per_page)¶

Set the default per_page url argument for paginated requests.

UserAgent(user_agent)¶

Set the user agent sent by PyGithub to GitHub API v3. Defaults to “jacquev6/PyGithub/2.0.0-alpha.3”.

Build()¶

Build and return a new instance of Github.

class PyGithub.Blocking._session.Session¶

Class representing a GitHub API v3 session.

Don’t create any instance of this class yourself. Use Builder to get a Github and then access its SessionedGithubObject.Session attribute (or on any returned object).

RateLimit¶

The last rate limit information received from GitHub (through headers x-ratelimit-...).

Type:RateLimits
OAuthScopes¶

The last oauth scopes information received from GitHub (through header x-oauth-scopes).

Type:None or list of string
AcceptedOAuthScopes¶

The last accepted oauth scopes information received from GitHub (through header x-accepted-oauth-scopes).

Type:None or list of string
PerPage¶

The value of the per_page argument sent in all paginated requests.

See also PaginatedList.

Type:None or int.

Base classes¶

class PyGithub.Blocking._base_github_object.SessionedGithubObject¶

You should not manipulate this class directly. It is public only to document its Session attribute, but this is an implementation detail. The Session attribute may be moved to another class or duplicated in concrete subclasses.

Session¶

You can use this shared attribute to access session-wide information, like your rate limit in RateLimit

class PyGithub.Blocking._base_github_object.UpdatableGithubObject¶

You should not manipulate this class directly. It is public only to document its update() method, but this is an implementation detail. The update() method may be moved to another class or duplicated in concrete subclasses.

update()¶

Makes a conditional request and updates the object.

Returns True if the the update was needed.

Exceptions¶

Error handling in PyGithub is done with exceptions. All exceptions derive (indireclty) from PyGithubException.

@todoSomeday generate the exception hierarchy graph

exception PyGithub.Blocking._exceptions.PyGithubException¶

Base class for all exceptions raised by PyGithub.

exception PyGithub.Blocking._exceptions.BadAttributeException¶

Raised when GitHub API v3 returns an attribute that doesn’t match the expected type or format.

exception PyGithub.Blocking._exceptions.ClientErrorException¶

Base class for exceptions raised by PyGithub when GitHub API v3 returns a 4XX HTTP status code.

exception PyGithub.Blocking._exceptions.ObjectNotFoundException¶

Raised by PyGithub when GitHub API v3 returns an unexpected 404 HTTP status code.

exception PyGithub.Blocking._exceptions.MethodNotAllowedException¶

Raised by PyGithub when GitHub API v3 returns a 405 HTTP status code. For example when PullRequest.merge() is called and PullRequest.meargeable was false.

exception PyGithub.Blocking._exceptions.ConflictException¶

Raised by PyGithub when GitHub API v3 returns a 409 HTTP status code. For example when there is a conflict in Repository.merge().

exception PyGithub.Blocking._exceptions.UnauthorizedException¶

Raised by PyGithub when GitHub API v3 returns a 401 HTTP status code.

exception PyGithub.Blocking._exceptions.ForbiddenException¶

Base class for exceptions raised by PyGithub when GitHub API v3 returns a 403 HTTP status code.

exception PyGithub.Blocking._exceptions.RateLimitExceededException¶

Raised by PyGithub when GitHub API v3 returns a 403 HTTP status code and the remaining rate limit is null.

exception PyGithub.Blocking._exceptions.UnprocessableEntityException¶

Base class for exceptions raised by PyGithub when GitHub API v3 returns a 422 HTTP status code.

exception PyGithub.Blocking._exceptions.ServerErrorException¶

Base class for exceptions raised by PyGithub when GitHub API v3 returns a 5XX HTTP status code.
