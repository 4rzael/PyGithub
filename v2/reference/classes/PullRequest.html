PullRequest.edit(title=None, body=None, state=None)¶

Calls the PATCH /repos/:owner/:repo/pulls/:number end point.

This is the only method calling this end point.

  • title – optional string
  • body – optional string or Reset
  • state – optional “closed” or “open”
